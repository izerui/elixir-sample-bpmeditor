# Help panel contents
bpmeditor.help.text = BPM エディターのサンプルでは、IBM ILOG Elixir Enterprise を使用して\
ビジネス・プロセス・ダイアグラム・エディターを作成する方法を示します。\n\
\n\
ビジネス・プロセス・ダイアグラムの作成を開始するには、BPM エレメントをパレットから作業域にドラッグ・アンド・ドロップします。\n\
2 つのエレメント間のリンクを作成するには、赤い三角形の接続ポイントを最初のエレメント側から 2 番目のエレメントの接続ポイントまでドラッグします。\n\
\n\
エレメントのテキストを編集するには、そのテキストをダブルクリックします。\n\
\n\
サンプル・ダイアグラム・ファイルは samples/bpmeditor/data ディレクトリーに入っています。\n\
\n\
ダイアグラムにフォーカスがあるときに、以下のホット・キーを使用できます。\n\
削除またはバックスペース: 選択済みのオブジェクトを削除します。\n\
Ctrl キー +「a」: すべてのオブジェクトを選択します。\n\
Ctrl キー +「c」: 選択済みのオブジェクトをコピーします。\n\
Ctrl キー +「g」: 選択済みのオブジェクトをサブプロセスとしてグループ化します。\n\
Ctrl キー +「j」: 選択済みの 2 つのノード間にリンクを作成します。\n\
Ctrl キー +「u」: 選択済みのサブプロセスからその親グラフにすべてのオブジェクトを移動して、サブプロセスを削除します。\n\
Ctrl キー +「v」: コピーしたオブジェクトを貼り付けます。\n\
Ctrl キー +「x」: 選択済みオブジェクトを切り取ります。\n\
F7: ダイアグラム内のオブジェクトでのキーボード・ナビゲーションを有効にします。F7 を 1 回押すと、ナビゲーションがノード全体を循環するように設定されます。 \
F7 をもう 1 回押すと、ナビゲーションが選択済みノードからこのノードに接続するリンクに進むように設定されます。\n\
F8: ダイアグラム内のキーボード・ナビゲーションを無効にします。\n\
矢印キー: キーボード・ナビゲーションが有効になっている場合、矢印キーを使用してオブジェクト間でフォーカスおよび選択を移動することができます。 \
シフト・キーを押しながら矢印キーを押すと、選択済みのノードが移動します。\n\
矢印キー: キーボード・ナビゲーションが無効になっている場合、矢印キーを使用してダイアグラムを水平または垂直に移動したり、選択済みオブジェクトを移動したりすることができます。\n\
Ctrl + 「+」キー: ズームインします。\n\
Ctrl + 「-」キー: ズームアウトします。\n\
\n\
オブジェクトのパレットにフォーカスがある場合は、以下のようになります。\n\
矢印キー: パレット内のノードを選択します。\n\
ENTER またはスペース: 選択済みオブジェクトをダイアグラムに作成します。\n\
シフト +「+」キー: パネルの領域を大きくします。\n\
シフト +「-」キー: パネルの領域を小さくします。\n\
\n\
プロパティー・パネルまたはヘルプ・パネルにフォーカスがある場合は、以下のようになります。\n\
シフト +「+」キー: パネルの領域を大きくします。\n\
シフト +「-」キー: パネルの領域を小さくします。
